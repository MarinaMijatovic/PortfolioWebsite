
#### Short project description
E-diary for math&programming school. The application enables records of students' attendance by subjects, lecturers' insight into the statistics of classes held, as well as the calculation of earnings and the calculation of students' expenses.
#### Technologies
Python
#### Note:
New version that should come in april, with new functionalities will be made in Flutter with Firebase. Desktop, mobile and
web application will be supported.

#### Approximate project time duration:
30h
#### Approximate updated version time duration
120h
